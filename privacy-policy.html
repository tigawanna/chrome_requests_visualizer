Privacy Policy for Chrome Requests Visualizer

Last updated: January 02, 2026

Important Notice: This extension does NOT collect, transmit, or store any personal data. All processing happens locally on your device. No data leaves your browser.

1. Overview

Chrome Requests Visualizer is a developer tool extension that visualizes network requests within your browser. This privacy policy explains our complete lack of data collection in compliance with GDPR and other privacy regulations.

2. Data Collection - NONE

We do NOT collect any personal data. We do NOT collect any usage data. We do NOT collect any analytics.

The extension operates entirely locally on your device and does not:

  • Collect personal information
  • Collect usage statistics
  • Use cookies or tracking technologies
  • Transmit data to external servers
  • Store data outside your browser

3. Local Processing Only

All extension functionality occurs locally within your browser:

  • Network request visualization processes data in memory
  • Any preferences are stored locally using Chrome's storage API
  • No external API calls or server communications
  • No data sharing with third parties

4. Required Permissions Explained

The extension requires Chrome permissions solely for local functionality:

  • "tabs": To identify active tabs for request monitoring
  • "webRequest" and "webRequestBlocking": Required by Chrome API to intercept and visualize network requests (data stays local)
  • "activeTab": To interact with the current webpage
  • "storage": To save your local preferences (colors, layout settings, etc.)

These permissions DO NOT grant the ability to transmit data externally.

5. GDPR Compliance

Under GDPR, you have the following rights. Since we collect no data:

  • Right to Access: We have no data to provide access to
  • Right to Erasure: We have no data to erase
  • Right to Rectification: We have no data to correct
  • Right to Restrict Processing: We perform no processing
  • Right to Data Portability: We have no data to port
  • Right to Object: We perform no processing to object to

6. Data Storage

Any user preferences (theme settings, layout choices) are stored exclusively in your browser using Chrome's local storage API. This data remains on your device and can be cleared at any time by:

  • Clearing browser data for the extension
  • Uninstalling the extension

7. Third-Party Services

This extension does not integrate with any third-party services, APIs, or external platforms.

8. Children's Privacy

Since we collect no data, we do not knowingly collect information from children or anyone else.

9. Changes to This Privacy Policy

We may update this privacy policy. The "Last updated" date at the top will reflect changes. Since we collect no data, changes will only clarify our non-collection practices.
