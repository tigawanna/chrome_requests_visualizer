Privacy Policy for Chrome Requests Visualizer

Last Updated: [Current Date]

Information Collection

Chrome Requests Visualizer is a developer tool that visualizes network requests from web pages. The extension:

  • Collects network request data ONLY from the browser's Developer Tools panel
  • All data processing happens locally in your browser
  • No data is sent to external servers
  • No personal information is collected

Permissions Explanation

The extension requires the following permissions:

  • "tabs" permission: To detect active tabs for request visualization
  • "webRequest" and "webRequestBlocking": To intercept and analyze network requests (required by Chrome for request visualization extensions)
  • "activeTab": To interact with the current webpage
  • "storage": To save your preferences locally

Data Storage

All data is stored locally in your browser using Chrome's storage API. No data is collected or transmitted to external servers.
